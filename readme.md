#   Multivariate-statistical-analysis

## 项目介绍
UESTC 2020级《多元统计与分析》课程内若与作业

## 主要部分
- 课程内容
1. 第一章：Python基础语法
2. 第二章：基本数据处理
3. 第三章：主成分分析
4. 第四章：因子分析(粗略)
5. 第五章：方差分析
6. 第六章：聚类分析
7. 第七章：回归分析

- 课程作业
1. 第一次作业：分析A股“地区板块”
2. 第二次作业：疫情对行业的影响
3. 第三次作业：美联储加息的影响
